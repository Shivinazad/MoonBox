# 🌑 moonbox

### Your social media command center.

**moonbox unifies performance tracking, real-time updates, and execution tools into one clean dashboard — so creators and small teams can plan, monitor, and act from a single workspace.**

<br/>

---

</div>

## The Problem

Managing social media across platforms means jumping between analytics tools, to-do lists, schedulers, and calendars — all separately. moonbox collapses that into one place.

---

## What moonbox does

#### 📊 Analytics Overview — know what's happening
See all your key metrics in one grid: Facebook shares, Instagram followers, LinkedIn views, engagement rate, and conversion status. A "last synced" timestamp keeps you confident the data is fresh. A live feed streams real-time activity updates from across your platforms as they happen.

#### ✅ Task Manager — act on what you find
Turn insights into action. If engagement dropped, create a task right there. Tasks persist, so nothing falls through the cracks.

#### 📅 Content Calendar — plan what's next
Month, week, and day views for planning content and campaigns. Add, edit, and delete events through a clean modal interface.

#### 💰 Revenue Overview — the bigger picture
A business snapshot with income, withdrawals, and a monthly revenue bar chart — for creators who track more than just likes.

#### 🔔 Notification Center — stay aware, not overwhelmed
A notification panel with preference controls for engagement alerts, mentions, sync updates, and performance signals. You decide what you hear about.

## User Journey

```
Landing page  →  Sign up / Log in  →  Analytics dashboard
                                            ↓
                               Tasks · Calendar · Revenue · Settings
```

---

## Auth & Data

- Secure login with bcrypt-hashed passwords
- User data stored in MongoDB
- Live analytics generated by backend and updated periodically
- Events stored in localStorage (no setup needed)
- Notification preferences saved per user

---

## Run Locally

```bash
git clone https://github.com/Shivinazad/social-media-dashboard.git
cd social-media-dashboard
npm install
npm start
```

Open `http://localhost:3000` in your browser.
